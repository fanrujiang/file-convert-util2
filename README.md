### 一、前言

目前支持：

1. `word` 转 `html`
2. `word` 转 `pdf`
3. `word` 转 `jpeg`
4. `word` 转 `png`
5. `html` 转 `word`
6. `html` 转 `pdf`
7. `html` 转 `png`
8. `doc` 转 `docx`
9. `excel` 转 `pdf`
10. `markdown` 转 `html`
11. `pdf` 转 `word`
